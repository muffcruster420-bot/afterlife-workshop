# Bedside SOP v0.1 (research only)

1. EEG 10-20, Fpz-Cz or Cz-Pz, ≥200 Hz, 2 min eyes-closed
2. Filter 0.5-70 Hz, notch 60 Hz
3. Compute bicoherence: 2-sec Hanning epochs, f1=15 Hz, f2=30 Hz (sum 45 Hz)
4. Report mean:
   - <0.3 = living baseline
   - >0.7 = peri-arrest pattern
   - 0.6-0.7 = check artifact (no stable occupancy expected)

Not diagnostic. Do not alter clinical care. Obtain consent.
