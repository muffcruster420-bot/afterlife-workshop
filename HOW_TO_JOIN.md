# How to Join

1. Read SAFETY.md
2. Listen to 45-Hz tone 3 minutes (or 528-Hz if 45 feels too much)
3. Journal what you notice — push, squeeze, calm, bird location
4. Open a GitHub Issue with tag `participant` — include city, age range, and notes
5. Optional: upload anonymized EDF if you have EEG

No payment. No readings. Data shared CC-BY 4.0.
