# The Shangraw Gap

**45 Hz bicoherence in 104 human brains**

| State | Bicoherence | n | Range |
|-------|-------------|---|-------|
| Waking | 0.304 ± 0.149 | 104 | 0.070 - 0.748 |
| Dying | 0.154 ± 0.345 | 5 | 0.000 - 0.771 |

**Finding:** The maximum dying brain (0.771) sits 3.1σ above the waking mean. The highest living subject reached 0.748 — 97% of the death value.

No gap. Just the far edge of human consciousness.

## Limitations

- **Dying cohort n=5: Only 1/5 brains (20%) exhibited surge to 0.771. Four PNAS validation cases showed 0.000 bicoherence, indicating the 0.771 signature is rare, not universal. The 0.771 value represents an exceptional case, not a population mean.
- **Waking cohort n=104**: Substantial for bicoherence analysis (typical EEG studies: n=20-50), but drawn from open datasets with variable recording conditions.

- **The 0.748 living maximum**: Represents 97% of the dying value and sits at 3.1σ. This suggests the dying state is extreme but not supernatural — it's the tail of the distribution.
